How to use Heroku with postgres:

Helpful page: 
1)Log in to heroku in the app directory using: heroku login (then enter your e-mail and password for heroku)
2)To see the list of apps you can access use: heroku list
3)To run app on heroku in browser use: heroku open --app "name of app" . Note: I'm not sure if that's running locally or running on the heroku server.
4)To establish a connection with the the heroku database using psql: heroku pg:psql HEROKU_POSTGRESQL_RED_URL --app dog-diabetes
